# Kermack-McKendrick-Double-Threshold-model
This project explores a modified Kermack–McKendrick epidemic model with cumulative exposure and a double threshold phenomenon, based on the work by Joan Ponce and Horst R. Thieme and Horst R. Thieme. 

The repository includes a Python implementation of the model introduced by Ponce and Thieme. The model captures how cumulative exposure leads to threshold-driven changes in system behavior.

The project also explores how this framework can be extended to other settings, including financial markets, where cumulative effects may lead to nonlinear and regime-like dynamics.
